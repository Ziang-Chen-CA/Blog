    # Reverse String (No.344)

    Write a function that reverses a string. The input string is given as an array of characters s by modifying the input array in-place (opens new window) with O(1) extra memory.

    My solution:

    class Solution:
    def reverseString(self, s: List[str]) -> None:
        i = 0
        j = len(s) - 1
        while i < j:
            s[i] ,s[j] = s[j], s[i]
            i += 1
            j -= 1
        return 
        """
        Do not return anything, modify s in-place instead.
        """

    # Reverse String (No.541)

    Given a string s and an integer k, reverse the first k characters for every 2k characters counting from the start of the string.

    If there are fewer than k characters left, reverse all of them. If there are less than 2k but greater than or equal to k characters, then reverse the first k characters and leave the other as original.

    My solution:

    class Solution:
    def reverseStr(self, s: str, k: int) -> str:
        i,j = 0, len(s) - 1
        s = list(s)
        for current_section in range(0, len(s), 2*k):
            s[current_section:current_section + k] = self.flip(s[current_section:current_section + k])
        return ''.join(s)
    def flip(self, s):
        i, j = 0, len(s) - 1
        while i < j:
            s[i], s[j] = s[j], s[i]
            i += 1 
            j -= 1
        return s

    # Reverse Each Word (No.151)

    Given an input string s, reverse the order of the words.

    Input: s = "  hello world  "
Output: "world hello"

    It is a must to get rid of all the space.

    class Solution:
    def reverseWords(self, s: str) -> str:
        s = s.strip().split()
        s = s[::-1]
        res = ' '.join(s)
        return res

    # KMP of Find the Index of the First Occurrence in a String (No.28)

    Given two strings needle and haystack, return the index of the first occurrence of needle in haystack, or -1 if needle is not part of haystack.

    The KMP is with O(sum of the sizes of two string).

    Here it goes like:

    class Solution:
    def strStr(self, haystack: str, needle: str) -> int:
        if len(needle) == 0:
            return 0
        next = [0]*len(needle)
        self.getNext(next, needle)
        common_length = 0
        for i in range(len(haystack)):
            while common_length > 0 and haystack[i] != needle[common_length]:
                common_length = next[common_length - 1] 
                #go to just the next after all the same letters
            if haystack[i] == needle[common_length]:
                common_length += 1
            if common_length == len(needle):
                return i - len(needle) + 1
        return -1

    def getNext(self, next: List[int], s:str) -> None:
        index = 0 # the index is to create the next
        next[0] = 0
        common_length = 0 # the initial should be 0
        for index in range(1, len(s)):# the next[0] is 0
            while common_length > 0 and s[index] != s[common_length]:
                common_length = next[common_length - 1]
            if s[index] == s[common_length]:
                common_length += 1
            next[index] = common_length

    In the getNext function, the method is to record the should-be-restart-point in the next[] array. Use index to scan and record. Then, the common length of next[0] is 0 cuz the first one letter got no pre and suffix.

    Then the classic part for the not-match: two kinds of while conditions that one for boundary control and the other is for on-going process until it hits the boundary or be able to be the same as earlier prefix. Note that The while loop inside makes the comparison pointer, common_length in the code, go back to the exact next one after the common pre and suffix by taking the value of next[common_length - 1].

    # Repeated Substring Pattern Only Confirmation (No.459)

    Given a string s, check if it can be constructed by taking a substring of it and appending multiple copies of the substring together.

    My solution is:

    class Solution:
    def repeatedSubstringPattern(self, s: str) -> bool:
        n = len(s)
        for i in range(1, n//2 + 1):
            if n % i == 0:
                number_of_substr = n // i
                new_str = number_of_substr * s[0:i]
                if new_str == s:
                    return True
        return False
